Code Quiz

Code Quiz

Code Quiz is an interactive web application that quizes the user on coding questions. It features a timer and a high score page. If a question is answered incorrectly, 15 seconds are deducted.


Tech/Frameworks used:
  • HTML
  • CSS (via Bootsrap)
  • Javascript
Features:

This application features a responsive web design, powered through Bootstrap, which allows it to adapt to multiple screen sizes. This app also features a timer and high score page, which pulls scores from local storage.

Day Planner

Day Planner

Day Planner is a simple planner application that allows the user to log plans for different hours of their work day.


Tech/Frameworks used:
  • HTML
  • CSS
  • JQuery
  • Moments.JS API
Features:

  • This app features time slots for standard business hours (9am-5pm) and includes an edit and save button, which will store the time and insput in 'localStorage,' and continue to be visible to the user after leaving and re-entering the page.
  • Each time slot will be color coded based on if it is a past, present or future hour
  • The top of the planner will display the current date.

    This weather application allows the user to input their city and receive current weather information in addition to a 5-day forecast.


    Tech/Frameworks used:
    • HTML
    • CSS
    • JQuery
    • OpenWeatherMap API
    Features:
    • This app uses the OpenWeather API to retrieve weather data.
    • Includes a search bar which allows user to access previous searches

      - Wind speed

      - UV index

      - Date

      - Humidity

    Tasty Trivia

    Tasty Trivia

    This app is a Trivia app, allowing users to test their knowledge on different topics while getting entertained.


    Tech/Frameworks used:
    • HTML
    • CSS via Bootstrap
    • JQuery
    • Node.JS
    • MySQL
    • Handlebars
    • Heroku
    Features
    • This app feautures a homepage, in which the user inputs their name, followed by a page that allows the user to select one of the four topics to play. The game is timed, and is followed by a score page.
    • This app also features a scoreboard, which pulls scores globally.
    • Our MySQL database was used to store our questions and answers for each of our topics, which we achieved by converting our data from Excel files to MySQL tables using a program called Excel2MySQL, and created the database itself using Sequelize.
